# Image Studio Backend

FastAPI backend for image generation, analysis, and model management using Stable Diffusion.

---

## Overview

This backend provides:

- Text-to-image generation
- Image analysis + prompt building
- Async job queue (generation jobs)
- Model installation and registry
- Runtime diagnostics (CPU/GPU info)
- Image serving

---

## Tech Stack

- Python 3.10+
- FastAPI
- PyTorch
- diffusers (Stable Diffusion / SDXL)
